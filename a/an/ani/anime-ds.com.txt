Information Collection and Use
Anime-DS.com is the sole owner of the infomation collected on our web site. Anime-DS.com collects information at several different points on our Web site.Information is collected for the purpose of tracking hits/views on a daily basis. A user must provide the clickstream information (internet protocol address, internet service provider, platform type, date/time stamp, operating system), http protocol information (last uri requested and browser type), unique identifiers (login id or other id), search terms (search text), transaction information (account transactions), user's name, user's birthdate, user's login information and user's gender (male or female). It is optional for the user to provide User's Name, User's Birthdate and User's Gender (Male or Female). We encourage users to provide optional information so we can provide a more personalized experience on our site. The reason we use this information is to complete the current transaction. We retain this information only as long as required to complete the action that required the information to be collected. Cookies
A cookie is a piece of data stored on the user's computer tied to information about the user. For the session ID cookie, once users close the browser, the cookie simply terminates. A persistent cookie is a small text file stored on the user's hard drive for an extended period of time. Persistent cookies can be removed by following Internet browser help file directions. You can set your browser to notify you to either accept or deny the cookie.
Our website uses cookies to enhance our site.
