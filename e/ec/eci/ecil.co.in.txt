Welcome to Electronics Corporation of India Limited
The ECILï¿½s website does not automatically capture or store personal information, other than information that is automatically logged about visitors and which is used to produce statistics about the use of the site. The following types of information are automatically collected and stored about any visit:
The Internet domain and Internet Protocol (IP) address from which a visitor accesses the site;
The type of Internet browser and the operating system of the computer visitor uses to access the site;
The date and time of visiting the site;
The pages visitor visited on the site;
If visitor linked to ECIL website from another Website, the address of that Website;
If linked to the site from a search engine, the address of that search engine and the search term used.
This information is collected by ECIL web server and is only used for system administration and to provide statistics which ECIL uses to evaluate the website.
The visitor will be asked for personal information if visitor wishs to use ECIL services such as Email. ECIL will only use the personal information provided to deliver the services visitor has requested. ECIL will not provide visitor information to any other organisation.
